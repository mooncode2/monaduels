**This is a Minecraft Java plugin that adds duels between players and parties to your server!**

**Features:**

Easy kit creation

Easy adding of arenas

Pull arenas

Editing kit layouts for players

Changing armor templates for donators only

Support & colors

Victory celebrations

And much more

**Commands:**

```
/duel NikName
```

Opens a GUI menu with a kit selection for the duel and sends a duel request.

```

/duel spectate
```

Opens a GUI menu with a selection of current duels to spectate.

```
/duel accept/deny
```

Accepts or declines a duel request.

```
/duel leave
```

Leaves a duel or mode. Observing a duel

```
/duel admin reload
```

Reloads the plugin configurations (does not recreate; to recreate, restart the server)

```
/duel admin createkit KitName
```

Creates a kit named KitName from items in your inventory, saves it to the plugin folder, and overwrites any existing kit.

```
/duel admin editkit KitName
```

Gives you a saved kit. This only gives you a saved kit. To save it, use /duel admin createkit KitName

```

/duel admin removekit KitName
```

Deletes a kit from the plugin folder.

```
/duel admin createarena ArenaName
```

Creates an arena template in the plugin. It can be modified later. The arena is created in the Multiverse world, which is defined in config.yml

```
/duel admin deletearena ArenaName
```

Removes an arena from the plugin

```
/duel admin setspawn ArenaName 1/2
```

Sets the player spawn rate in the arena

```
/duel admin setspectatorspawn ArenaName
```

Sets the spectator spawn rate in the arena
